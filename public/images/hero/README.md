# Hero Slider — Home

Upload 4 gambar slide hero halaman Home ke folder ini dengan nama persis:

- `slide-1.jpg`
- `slide-2.jpg`
- `slide-3.jpg`
- `slide-4.jpg`

Slide berganti otomatis tiap 4 detik (lihat `src/app/pages/Home.tsx`).

Rekomendasi: gambar landscape, lebar minimal 1600px, format `.jpg`.
